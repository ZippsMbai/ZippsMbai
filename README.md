## 📂 Projects

### 🛡️ [infra-patch-tracker](./infra-patch-tracker)
> Flask web app · Python · JSON storage

A lightweight patch management tracker for IT teams who need auditable patch records without a full enterprise ITSM platform. Tracks OS patches and CVEs across infrastructure endpoints, with a compliance dashboard, per-system reporting, and a JSON API.

**Runs on:** Replit · Gitpod · Any Python environment  
**Try it:** Clone → `pip install flask` → `python app.py` → open browser → click **Load Demo**

---

### 🔍 [iam-audit-tool](./iam-audit-tool)
> Python CLI · Zero dependencies · Standard library only

A command-line IAM configuration auditor that detects common misconfigurations: stale accounts, admins without MFA, overprivileged service accounts, shared accounts, and expired passwords. Outputs colour-coded findings with risk scores and remediation steps.

**Runs on:** Replit · Any Python 3.8+ environment  
**Try it:** Clone → `python iam_audit.py` (no install needed)

```bash
python iam_audit.py --severity high --format json
```

### 🔐 [siem-log-dashboard](./siem-log-dashboard)
> Single-file HTML/JS · No backend · No install

A browser-based SIEM event dashboard with live simulation of security events (AUTH, NETWORK, ENDPOINT, DLP, SYSTEM). Features real-time stats, animated charts, a searchable/filterable log, and critical alert modals. Built to prototype and demo SIEM visualisations without infrastructure access.

**Runs on:** Any browser — just open `index.html`  
**Try it:** Clone → double-click `index.html`
